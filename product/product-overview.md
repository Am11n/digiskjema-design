# Digiskjema

## Description

Digiskjema er en digital skjemaplattform for norske kommuner som gjør det mulig å opprette, administrere og håndtere digitale skjemaer for kommunikasjon med innbyggere og virksomheter. Plattformen tilbyr et fullverdig skjemabyggerverktøy med funksjoner for design, publisering, innsending og behandling av skjema. Løsningen støtter både enkelte kommuner og felles drift for flere kommuner gjennom en multi-tenant arkitektur.

## Problems & Solutions

### Problem 1: Kompleks skjemahåndtering
Gjør det enkelt for kommuner å opprette og håndtere digitale skjemaer uten avansert teknisk kompetanse

### Problem 2: Manglende tilgjengelighet
Sikrer at alle skjema er tilgjengelige for brukere med ulike behov og funksjonsvariasjoner

### Problem 3: Manuell prosesshåndtering
Automatiserer innsending, behandling og arkivering av skjema

### Problem 4: Manglende integrasjon
Gir mulighet for kobling mot eksisterende kommunale systemer

### Problem 5: Sikkerhet og personvern
Sikrer beskyttelse av personopplysninger og sikker overføring

### Problem 6: Multi-tenant behov
Gir mulighet for både enkeltkommunale og felles driftsmodeller

### Problem 7: Ulike innsendingskanaler
Støtter ulike måter å sende inn skjema på (ID-porten, anonymt, etc.)

## Key Features

- Fullverdig skjemabygger med drag-and-drop funksjonalitet
- Multi-tenant arkitektur for enkelt og felleskommunal drift
- WCAG 2.1 AA tilgjengelighet som standard
- ID-porten og D-nummer autentisering
- Integrering med Elements (sak/arkivsystemer)
- API-first design for tredjepartsintegrering
- Malbibliotek og gjenbrukbare komponenter
- Analyseverktøy og bruksstatistikk
- Full logging og revisjonsfunksjonalitet
- Norsk bokmål og nynorsk med grunnlag for flerspråklighet
