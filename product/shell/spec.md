# Application Shell Specification

## Overview

The Digiskjema application uses a sidebar navigation layout with header, sidebar, and main content areas. This provides clear navigation for municipal employees managing forms while maintaining focus on the primary content.

## Navigation Structure

- Skjema → Form listing and filling
- Bygg skjema → Form builder
- Malmaker → Template gallery
- Kategorier → Category management
- Leietakarar → Multi-tenant management
- Betingelser → Conditional logic
- Analyser → Analytics
- Språk → Language management
- Tilgjenge → Accessibility features
- Admin → System administration

## Layout Pattern

Sidebar navigation layout with:
- Fixed header at top containing logo and user menu
- Persistent sidebar navigation on left side
- Main content area that scrolls independently
- Responsive design that collapses to mobile drawer on small screens
- Clean, professional appearance following Digdir Design System guidelines
