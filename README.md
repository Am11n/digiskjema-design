# Digiskjema Design

Digital form system for the Norwegian public sector. This project provides a comprehensive platform for creating, managing, and processing digital forms with focus on accessibility, multilingual support, and compliance with Norwegian public sector standards.

## Overview

Digiskjema Design is a complete solution for digital form handling in Norwegian public sector organizations. The system includes:

- **Form Creation**: Intuitive form builder for creating complex forms with conditional logic
- **Form Management**: Administrative tools for managing templates, categories, tenants, and access
- **Form Processing**: Secure submission and processing system with analytics
- **Accessibility**: Full compliance with WCAG 2.1 AA standards
- **Multilingual Support**: Norwegian Bokm√•l, Norwegian Nynorsk, and English support
- **Analytics**: Insights dashboard for understanding usage patterns and accessibility

## Features

### Form Building
- Drag-and-drop form builder
- Conditional logic and validation rules
- Integration with public sector data sources
- Template management system

### Administrative Tools
- Tenant management for different organizations
- Category and template organization
- Role-based access control

### Analytics & Insights
- Usage statistics and KPIs
- Accessibility compliance monitoring
- Language usage analytics
- Form performance metrics

### Technical Features
- Responsive design for all devices
- WCAG 2.1 AA compliance
- Norwegian language support (nb, nn, en)
- Secure data handling
- API-first architecture

## Architecture

The system follows a modular architecture with distinct sections:

- **Form Section**: Handles form listing, filling out, and submission
- **Form Builder**: Tools for creating and publishing forms
- **Administration**: Management of templates, categories, tenants, and access control
- **Insights**: Analytics and monitoring tools

## Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn package manager

### Installation
1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

### Development
- The project uses React 19 with TypeScript
- Tailwind CSS for styling
- Lucide React for icons
- Vite for build tooling

## Sections

### Form (Skjema)
Complete form experience from listing to submission, including accessibility and authentication features.

### Form Builder (Skjemabygger)
Full-featured form builder with logic, validation, and publishing capabilities.

### Administration (Administrasjon)
Administrative functions for managing templates, categories, tenants, conditions, and access control.
